## üèÅ Usage

Clone the repository and go to the root directory

```bash
$ git clone https://github.com/viniciusgugelmin/solid

$ cd solid
```

Run the command to create the environment variables files

```bash
$ npm run setup
```

Run the application with docker:

```bash
$ docker-compose up
```
